# Stopwatch Web Application

A simple, interactive stopwatch with start, pause, reset, and lap timing features.

## Features

- **Start** - Begin timing
- **Pause** - Stop timing temporarily
- **Reset** - Reset to 00:00:00
- **Lap** - Record time intervals
- **Clear Laps** - Remove all lap records

## How to Use

1. Click **Start** to begin the stopwatch
2. Click **Lap** to record current time
3. Click **Pause** to stop the timer
4. Click **Reset** to clear everything
5. Click **Clear All Laps** to remove lap history

## How to Run

1. Download all files
2. Keep the folder structure as shown above
3. Open `index.html` in any web browser
